# Coding Plan

Use this skill when a task requires code changes.

1. Read the relevant files before editing.
2. Keep changes scoped to the requested behavior.
3. Add or update tests for behavior that can regress.
4. Run the smallest useful verification command.
5. Summarize what changed and what was verified.
